
# CampusMate - Study Task Manager

## Overview

CampusMate is an Android mobile application developed to help students manage their academic activities efficiently. The application provides task management functionality along with useful student-focused tools such as weather information and book search to improve productivity and organization.

The application was developed using Java and Android Studio with a simple, user-friendly interface designed for mobile devices.

---

## Features

### Task Management

* Create study tasks
* Edit existing tasks
* Delete tasks
* Track pending and completed tasks
* Organize daily academic activities

### Weather Information

* View current weather conditions
* Search weather by location
* Retrieve weather information using the Open-Meteo API

### Book Search

* Search books using the Open Library API
* View book details
* Save favorite books for future reference

### User-Friendly Interface

* Simple navigation
* Mobile-friendly design
* Easy access to application features

## How to Run

### Clone Repository

```bash
git clone https://github.com/asirilakshan/Campus-Mate.git
```

### Open Project

1. Open Android Studio
2. Select **Open Project**
3. Choose the project folder

### Build Project

Allow Gradle to download dependencies and sync the project.

### Run Application

1. Connect an Android device or launch an emulator
2. Click the **Run** button in Android Studio
3. Install and launch the application
